## 常用命令

```
hexo new '文章名'
hexo clean
hexo generate 
hexo s          //预览
hexo deploy    
```

## 将本地文件推送至“HEXO”分支

```
git add .
git commit –m "描述xxxx"
git push 
```

## 在另一设备更新

```
git pull
```
